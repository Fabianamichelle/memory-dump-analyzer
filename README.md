# Memory Dump String Frequency Analyzer
A Python tool that scans large binary memory dump files and extracts all strings between 5–15 characters, counts their frequency, and outputs a formatted report.
Features:

### Chunk-based processing — handles large files without memory issues
### Regex-based string extraction from raw binary data
### Frequency report saved as a formatted text file

## Usage:
```bash
bashpython analyzer.py
# Enter path to your .raw file when prompted
```
Technologies: Python, Regular Expressions, Collections, PrettyTable
